Welcome to GitHub Pages.

Going Dark is a Virtual Reality Space Shooter developed in Unity3D using the Oculus Rift and Leap Motion. The player progresses through several levels completing missions which include shooting asteroids, collecting loot, killing enemies and annihilating bosses.
